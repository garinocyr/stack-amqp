Docker stack for AMQP purpose
- Uses rabbitmq as AMQP broker (see https://www.rabbitmq.com/)

### Usage

* Start the stack
  * From local git repository

```console
developer@host:~/.../stack-amqp$ docker-compose up -d
```

* Stop the stack
  * From local git repository

```console
developer@host:~/.../stack-amqp$ docker-compose down
```
